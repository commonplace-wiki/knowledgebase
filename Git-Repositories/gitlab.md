GitLab is not supported yet. Commonplace currently works with [GitHub](/Git-Repositories/github.md) only; the provider is detected from the host in `GIT_REPO`, and GitLab hosts are not recognized.

Support for GitLab (sign-in via GitLab OAuth, permissions from project membership, commits attributed to the GitLab user) is on the roadmap. If you need it, follow or upvote the corresponding issue in the [Commonplace repository](https://github.com/commonplace-wiki/commonplace/issues).

Until then, a mirror setup can bridge the gap: keep the wiki repository on GitHub and mirror it to GitLab, or the other way around with GitHub as the writable side.
