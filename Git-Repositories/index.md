Commonplace stores all content in a Git repository and detects the provider from the host in `GIT_REPO`.

* [GitHub](/Git-Repositories/github.md) — fully supported, the primary backend.
* [GitLab](/Git-Repositories/gitlab.md) — planned.
* [Local Git](/Git-Repositories/local-git.md) — planned.
